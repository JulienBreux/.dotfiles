## Getting started

### Install brew

```bash
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
```

### Execute brew bundle install

```bash
brew bundle
```

### Run post-install

```bash
task setup
```
